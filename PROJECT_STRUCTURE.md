# Структура проекта GPON Generator

## Файловая структура

```
├── index.html          # Основной HTML
├── style.css           # Стили
├── main.js             # Основной JS
├── entry.js            # Модульный загрузчик (требует type="module")
├── adsl.js             # Конфигурации ADSL устройств
├── fttb.js             # Конфигурации FTTB устройств
├── gpon.js             # Конфигурации GPON устройств
├── sib_templates.js    # Шаблоны Сибири (ADSL/FTTB) из XLSX
└── (js/ удалён)         # Убрана дублирующая модульная версия логики
```

## Использование

### Основная версия (main.js)
```html
<script src="main.js"></script>
```

### Модульная версия (entry.js)
```html
<script type="module" src="entry.js"></script>
```

## Примечание
- `entry.js` теперь просто подгружает `main.js`, чтобы оставить опцию `type="module"` без дублирования кода.

## Добавление нового устройства

1. Добавить порт в `main.js` → `defaultPorts`
2. Добавить шаблон в `adsl.js`/`fttb.js`/`gpon.js`
3. Добавить команды диагностики в соответствующий файл устройств

## Добавление нового филиала Сибири

1. Добавить в `main.js` → `sibConfigs`
2. Добавить шаблоны в `gpon.js`
